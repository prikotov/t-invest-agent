# Ticker Analysis

## Когда использовать

- Оценка инвестиционной привлекательности тикера
- Проверка фундаментальных метрик
- Анализ ценовой динамики

## Как использовать

**Шаг 1:** Рыночные данные

```bash
./vendor/bin/moex security:trade-data {TICKER}
./vendor/bin/moex security:specification {TICKER}
```

`moex security:trade-data:`

| Опция  | Описание                        |
|--------|---------------------------------|
| TICKER | Тикер инструмента (позиционный) |

Поля:

| Поле   | Описание       |
|--------|----------------|
| last   | Последняя цена |
| open   | Цена открытия  |
| high   | Максимум       |
| low    | Минимум        |
| volume | Объём торгов   |

`moex security:specification:`

| Поле       | Описание        |
|------------|-----------------|
| isin       | ISIN код        |
| type       | Тип инструмента |
| list_level | Уровень списка  |
| issue_size | Размер выпуска  |

**Шаг 2:** Фундаментальный анализ

```bash
./vendor/bin/skill instruments:fundamentals {TICKER}
```

| Опция   | Описание                    |
|---------|-----------------------------|
| tickers | Тикеры (позиционные, через пробел) |

Поля:

| Поле           | Описание               |
|----------------|------------------------|
| pe             | P/E мультипликатор     |
| pb             | P/B мультипликатор     |
| roe            | Return on Equity       |
| dividend_yield | Дивидендная доходность |

**Шаг 3:** Исторические данные

```bash
./vendor/bin/skill market:candles {TICKER} --from 2024-01-01
```

| Опция   | Описание              |
|---------|-----------------------|
| ticker  | Тикер (позиционный)   |
| --from  | Дата начала           |
| --to    | Дата окончания        |

**Шаг 4:** Новости

```bash
./vendor/bin/news news:fetch --ticker {TICKER}
```

| Опция    | Описание         | По умолчанию |
|----------|------------------|--------------|
| --ticker | Фильтр по тикеру | —            |

## Результат

**{TICKER}: [Оценка]**

| Поле     | Описание          |
|----------|-------------------|
| Метрика  | Название метрики  |
| Значение | Числовое значение |
| Оценка   | ✓/✗               |

**Ценовая динамика:**

- Текущая: цена
- Период: изменение

**Новости:**

- [список заголовков]

**Рекомендация:** BUY / HOLD / SELL

Не является инвестиционной рекомендацией. {TICKER}
