# Corporate Events Calendar

## Когда использовать

- Поиск предстоящих дивидендов
- Календарь отчётностей эмитентов
- События по облигациям (купоны, погашения, оферты)
- Планирование сделок вокруг событий

## Как использовать

**Шаг 1:** Дивиденды по инструменту

```bash
./vendor/bin/t-invest events:dividends --ticker=SBER
./vendor/bin/t-invest events:dividends --ticker=SBER --from=2025-01-01 --to=2025-12-31
```

Опции:

| Опция    | Описание                    | По умолчанию    |
|----------|-----------------------------|-----------------|
| --ticker | Тикер инструмента           | — (обязательно) |
| --from   | Начало периода (YYYY-MM-DD) | —               |
| --to     | Конец периода (YYYY-MM-DD)  | —               |

Поля:

| Поле         | Описание                      |
|--------------|-------------------------------|
| dividendNet  | Размер дивиденда (нетто)      |
| paymentDate  | Дата выплаты                  |
| recordDate   | Дата отсечки                  |
| lastBuyDate  | Последний день для покупки    |
| dividendType | Тип (Regular, Special)        |
| yieldValue   | Доходность (%)                |

**Шаг 2:** Календарь отчётностей

```bash
./vendor/bin/t-invest events:reports --ticker=SBER
./vendor/bin/t-invest events:reports --ticker=GAZP --from=2025-01-01 --to=2025-12-31
```

Опции:

| Опция    | Описание          | По умолчанию    |
|----------|-------------------|-----------------|
| --ticker | Тикер инструмента | — (обязательно) |
| --from   | Начало периода    | —               |
| --to     | Конец периода     | —               |

Поля:

| Поле       | Описание                     |
|------------|------------------------------|
| reportDate | Дата публикации              |
| periodNum  | Номер квартала (1-4)         |
| periodYear | Год отчётного периода        |
| periodType | Тип (Q1, Q2, Q3, Q4, Annual) |

**Шаг 3:** События по облигациям

```bash
./vendor/bin/t-invest events:bonds --ticker=SU26238RMFS
./vendor/bin/t-invest events:bonds --ticker=RU000A1038V6 --type=CPN
```

Опции:

| Опция    | Описание        | По умолчанию    |
|----------|-----------------|-----------------|
| --ticker | Тикер облигации | — (обязательно) |
| --type   | Тип события     | все             |
| --from   | Начало периода  | —               |
| --to     | Конец периода   | —               |

Типы событий (--type):

| Значение | Описание    |
|----------|-------------|
| CPN      | Купон       |
| CALL     | Оферта      |
| MTY      | Погашение   |
| CONV     | Конвертация |

Поля:

| Поле               | Описание                  |
|--------------------|---------------------------|
| eventDate          | Дата события              |
| eventNumber        | Номер события             |
| eventType          | Тип события               |
| payOneBond         | Выплата на одну облигацию |
| couponPeriod       | Период купона (дни)       |
| couponInterestRate | Ставка купона (%)         |

**Дивиденды:**

| Тикер | Дивиденд | Отсечка    | Выплата    | Доходность |
|-------|----------|------------|------------|------------|

**Отчётности:**

|-------|--------|------------|-----------|
| SBER  | Q1     | 2025-05-15 | 1Q2025    |
| GAZP  | Annual | 2025-04-25 | FY2024    |

**Облигации:**

| Тикер       | Тип    | Дата       | Сумма   |

## Примечания

- Для дивидендов: дата отсечки = recordDate, T+1 режим расчётов
- Для отчётностей: публикация обычно за 1-2 месяца до конца квартала
- Для облигаций: купоны выплачиваются по графику эмитента
