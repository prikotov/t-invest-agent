# Equity Research

Профессиональная аналитика по российским компаниям: фундаментальный анализ, оценка отчётности, дивидендные прогнозы, инвестиционные стратегии.

---

## Назначение

Использовать для получения **качественной аналитики** от профессиональных управляющих компаний и аналитиков:
- Глубокий анализ финансовой отчётности
- Оценка бизнес-модели и конкурентных преимуществ
- Прогнозы дивидендов и справедливой стоимости
- Инвестиционные рекомендации

Отличие от других skills:
- `news-deep` — информационные поводы, новости компаний
- `social-sentiment` — настроения инвесторов на форумах
- `equity-research` — профессиональная аналитика (фундаментальный анализ)

## Источники

### 1. Arsagera (БлогоФорум)

**URL:** `https://bf.arsagera.ru/`

Управляющая компания с глубоким фундаментальным анализом. Анализируют отчётность, оценивают качество бизнеса, пишут развёрнутые выводы.

**Структура:**
- Главная страница содержит темы по отраслям
- Каждая компания имеет страницу с тикером: `https://bf.arsagera.ru/{отрасль}/{тикер}/`
- Статьи: "Итоги 2025 г.", "Итоги 9 мес. 2025 г." и т.д.

**Отрасли:**
- `dobycha_pererabotka_nefti_i_gaza` — нефть и газ
- `finansovyj_sektor` — финансы
- `potrebitelskij_sektor` — потребительский сектор
- `svyaz_telekommunikacii_i_novye_tehnologii` — телеком и IT
- `mashinostroenie` — машиностроение
- `stroitelstvo_nedvizhimost` — недвижимость
- `transport` — транспорт
- `proizvodstvo_mineralnyh_udobrenij` — удобрения
- `prochie` — прочие

**Как искать:**

1. Прямой переход на страницу компании (если известна отрасль):
   ```
   https://bf.arsagera.ru/finansovyj_sektor/sberbank_rossii/
   https://bf.arsagera.ru/dobycha_pererabotka_nefti_i_gaza/lukojl/
   https://bf.arsagera.ru/svyaz_telekommunikacii_i_novye_tehnologii/yandex_nv_yndx/
   ```

2. Поиск через главную страницу:
   - webfetch `https://bf.arsagera.ru/`
   - Найти тикер/название компании в списке
   - Перейти по ссылке на страницу компании

3. Поиск через DuckDuckGo (если отрасль неизвестна):
    ```
    https://duckduckgo.com/html/?q=site%3Abf.arsagera.ru+{TICKER}
    ```
    **Важно:** DuckDuckGo может блокировать ботов капчей.

**Что анализируют:**
- Выручка и её динамика
- Операционная и чистая прибыль
- Долговая нагрузка
- Рентабельность
- Дивидендная политика
- Риски и перспективы

### 2. ДоходЪ

**URL:** `https://www.dohod.ru/`

Управляющая компания с аналитическими сервисами и исследованиями. Широкое покрытие эмитентов, дивидендный календарь, анализ акций и облигаций.

**Ключевые разделы:**

| Раздел           | URL                                          | Назначение                          |
|------------------|----------------------------------------------|-------------------------------------|
| Исследования     | `https://www.dohod.ru/analytic/research`     | Статьи аналитиков по эмитентам      |
| Дивиденды        | `https://www.dohod.ru/ik/analytics/dividend` | Календарь, прогнозы, история выплат |
| Анализ акций     | `https://www.dohod.ru/ik/analytics/share`    | Рейтинги, P/E, потенциал роста      |
| Анализ облигаций | `https://www.dohod.ru/analytic/bonds`        | Подбор облигаций по параметрам      |
| Котировки        | `https://www.dohod.ru/ik/analytics/stockmap` | Таблицы доходности по отраслям      |

**Как искать:**

1. Поиск по сайту через DuckDuckGo:
    ```
    https://duckduckgo.com/html/?q=site%3Adohod.ru+{TICKER}
    ```

2. Прямой доступ к разделам:
   - webfetch `https://www.dohod.ru/analytic/research?category=dividendyi` — дивидендные исследования
   - webfetch `https://www.dohod.ru/analytic/research?category=strategii` — стратегии

**Что доступно:**
- Прогнозы дивидендов и дат закрытия реестров
- Оценка справедливой стоимости
- Индекс стабильности дивидендов (DSI)
- Рейтинги акций по потенциалу роста
- Фундаментальные мультипликаторы (P/E, P/BV)

### 3. Вредный Инвестор (Telegram)

**URL:** `https://t.me/harmfulinvestor`

Telegram-канал с независимой аналитикой российского рынка. Автор публикует обзоры отчётности, комментарии по дивидендам, разборы компаний.

**Особенности:**
- Краткие и ёмкие обзоры отчётности
- Критический взгляд на корпоративные события
- Оценка качества менеджмента и корпоративного управления
- Комментарии по дивидендной политике

**Как использовать:**
- Искать по названию компании или тикеру в канале
- Смотреть закреплённые сообщения с обзорами
- Использовать как дополнение к фундаментальному анализу

**Что анализирует:**
- Качество отчётности и прозрачность
- Адекватность дивидендной политики
- Риски корпоративного управления
- Сравнение с конкурентами

## Алгоритм работы

### Шаг 1: Определить задачу

- **Анализ отчётности** → Arsagera (итоги квартала/года) + Вредный Инвестор
- **Дивиденды** → ДоходЪ (календарь, прогнозы) + Arsagera
- **Оценка справедливой стоимости** → ДоходЪ (анализ акций)
- **Корпоративное управление** → Вредный Инвестор
- **Инвестиционная идея** → Все источники

### Шаг 2: Получить данные

1. **Arsagera:**
   - webfetch главную страницу `https://bf.arsagera.ru/`
   - Найти компанию по тикеру или названию
   - Перейти на страницу компании
   - Прочитать последнюю статью по итогам отчётности

2. **ДоходЪ:**
    - webfetch `https://duckduckgo.com/html/?q=site%3Adohod.ru+{TICKER}`
    - Выбрать релевантные статьи
    - Прочитать исследования

3. **Вредный Инвестор:**
   - webfetch `https://t.me/s/harmfulinvestor?q={TICKER}`
   - Найти посты по компании
   - Извлечь критические замечания и выводы

### Шаг 3: Синтез и выводы

Объединить данные из источников:
- Факты из отчётности (Arsagera)
- Прогнозы и оценки (ДоходЪ)
- Сравнить с рыночной ценой
- Сформировать вывод

## Типовые сценарии

### Сценарий 1: Анализ компании перед покупкой

```
Запрос: "Проанализируй SBER как инвестицию"

Действия:
1. Arsagera: https://bf.arsagera.ru/finansovyj_sektor/sberbank_rossii/
   → Прочитать "Итоги 2025 г."
   → Оценить финансовое состояние, рентабельность, рост

2. ДоходЪ: site:dohod.ru SBER
   → Найти исследования по Сбербанку
   → Оценить прогноз дивидендов, целевую цену

3. Синтез:
   - Качество бизнеса: ...
   - Финансовое состояние: ...
   - Дивидендная доходность: ...
   - Риски: ...
   - Вывод: покупать/держать/продавать
```

### Сценарий 2: Оценка дивидендов

```
Запрос: "Какие дивиденды ожидать от Лукойла?"

Действия:
1. ДоходЪ: https://www.dohod.ru/ik/analytics/dividend
   → Найти LKOH в календаре
   → Прочитать прогноз дивидендов

2. Arsagera: https://bf.arsagera.ru/dobycha_pererabotka_nefti_i_gaza/lukojl/
   → Прочитать последнюю статью
   → Оценить способность платить дивиденды

3. Вывод: прогноз дивидендов, дата отсечки, доходность
```

### Сценарий 3: Сравнение компаний в отрасли

```
Запрос: "Сравни Сбербанк и ВТБ по фундаментальным показателям"

Действия:
1. Arsagera:
   - https://bf.arsagera.ru/finansovyj_sektor/sberbank_rossii/
   - https://bf.arsagera.ru/finansovyj_sektor/bank_vtb/

2. Извлечь ключевые метрики из последних статей:
   - ROE, ROA
   - Стоимость риска
   - Рентабельность
   - Темпы роста

3. Сравнительная таблица с выводами
```

### Сценарий 4: Анализ после отчётности

```
Запрос: "Что показал отчёт Яндекса за 2025 год?"

Действия:
1. Arsagera: https://bf.arsagera.ru/svyaz_telekommunikacii_i_novye_tehnologii/yandex_nv_yndx/
   → Найти статью "Итоги 2025 г."
   → Извлечь ключевые выводы

2. ДоходЪ: site:dohod.ru Яндекс YNDX
   → Найти реакцию аналитиков

3. Резюме: выручка, прибыль, драйверы роста, прогноз
```

## Ограничения

- Оба источника покрывают преимущественно российский рынок
- Arsagera — глубокий анализ, но не все компании покрыты
- ДоходЪ — требует регистрации для полного доступа к сервисам
- Обновления выходят после публикации отчётности (с задержкой)
