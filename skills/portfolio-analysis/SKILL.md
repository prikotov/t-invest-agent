# Portfolio Analysis

## Когда использовать

- Оценка текущих позиций
- Анализ распределения активов
- Диагностика портфеля

## Как использовать

**Шаг 1:** Получить позиции

```bash
    ./vendor/bin/skill portfolio:positions
```

Опции:

| Опция    | Описание      | По умолчанию |
|----------|---------------|--------------|
| --format | Формат вывода | table        |

Поля:

| Поле     | Описание          |
|----------|-------------------|
| ticker   | Тикер инструмента |
| name     | Название          |
| quantity | Количество        |
| price    | Текущая цена      |
| value    | Стоимость позиции |
| weight   | Доля в портфеле   |
| profit   | Прибыль/убыток    |

**Шаг 2:** Для каждой позиции > 5% веса — анализ

```bash
    ./vendor/bin/moex security:trade-data {TICKER}
    ./vendor/bin/news news:fetch --ticker {TICKER}
    ./vendor/bin/skill analyze:technical --ticker={TICKER}
```

`moex security:trade-data:`

| Опция  | Описание                        |
|--------|---------------------------------|
| TICKER | Тикер инструмента (позиционный) |

Поля: `last`, `open`, `high`, `low`, `volume`

`news news:fetch:`

| Опция    | Описание         | По умолчанию |
|----------|------------------|--------------|
| --ticker | Фильтр по тикеру | —            |
| --limit  | Лимит записей    | все          |

`skill analyze:technical:`

| Опция    | Описание          |
|----------|-------------------|
| --ticker | Тикер инструмента |

Поля: `trend`, `rsi`, `macd`, `signals`

**Шаг 3:** Сформировать сводку с рекомендациями
