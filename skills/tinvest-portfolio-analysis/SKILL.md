# Portfolio Analysis

## Когда использовать

- Оценка текущих позиций
- Анализ распределения активов
- Диагностика портфеля

## Как использовать

**Шаг 1:** Получить позиции

```bash
./vendor/bin/t-invest portfolio:show
```

Опции:

| Опция          | Описание               | По умолчанию |
|----------------|------------------------|--------------|
| -t, --ticker   | Фильтр по тикеру       | —            |

Поля:

| Поле     | Описание          |
|----------|-------------------|
| ticker   | Тикер инструмента |
| type     | Тип инструмента   |
| quantity | Количество        |
| avg_price| Средняя цена      |
| yield    | Прибыль/убыток    |
| price    | Текущая цена      |

**Шаг 2:** Для каждой позиции > 5% веса — анализ

```bash
./vendor/bin/moex security:trade-data {TICKER}
./vendor/bin/news news:fetch --ticker {TICKER}
./vendor/bin/t-invest instruments:fundamentals {TICKER}
```

`moex security:trade-data:`

| Опция  | Описание                        |
|--------|---------------------------------|
| TICKER | Тикер инструмента (позиционный) |

Поля: `last`, `open`, `high`, `low`, `volume`

`news news:fetch:`

| Опция    | Описание         | По умолчанию |
|----------|------------------|--------------|
| --ticker | Фильтр по тикеру | —            |

`t-invest instruments:fundamentals:`

| Опция   | Описание                    |
|---------|-----------------------------|
| tickers | Тикеры (позиционные, через пробел) |

**Шаг 3:** Сформировать сводку с рекомендациями

## Результат

**Портфель: Сводка**

| Поле         | Описание                 |
|--------------|--------------------------|
| Тикер        | Тикер инструмента        |
| Вес          | Доля в портфеле          |
| Цена         | Текущая цена             |
| Изм          | Изменение за период      |
| Рекомендация | HOLD / BUY MORE / REDUCE |

**Важные новости:**

- {TICKER}: [новость]
