# Social Sentiment Analysis

Анализ настроений розничных инвесторов и трейдеров. Показывает "народное мнение" по акции/компании.

## Когда использовать

- Оценка настроений перед принятием решения
- Понимание бычьего/медвежьего настроения толпы
- Поиск контрарных сигналов (слишком оптимистичны = продаём)
- Проверка репутации компании у клиентов
- Поиск слухов и инсайдов

## Источники

### Форумы

| Источник | URL | Что даёт |
|----------|-----|----------|
| SmartLab | `https://smart-lab.ru/forum/{TICKER}/` | Форум инвесторов, обсуждения, факторы роста/падения |
| MFD | `https://mfd.ru/forum/` | Форум трейдеров, популярные темы, котировки |
| Banki.ru форум | `https://www.banki.ru/forum/` | Финансовый форум, раздел "Рынок ценных бумаг" |

### Отзывы

| Источник | URL | Что даёт |
|----------|-----|----------|
| Banki.ru отзывы | `https://www.banki.ru/services/responses/bank/{slug}/` | Отзывы клиентов банков |

## Как использовать

### Шаг 1: SmartLab — форум инвесторов

**Шаг 1.1:** Открыть форум по тикеру

```
https://smart-lab.ru/forum/{TICKER}/
```

Примеры:
```
https://smart-lab.ru/forum/SBER/
https://smart-lab.ru/forum/GAZP/
https://smart-lab.ru/forum/LKOH/
```

**Шаг 1.2:** Извлечь из страницы

| Элемент | Где искать | Что анализировать |
|---------|------------|-------------------|
| Обсуждения | Основная лента | Тональность постов, аргументы |
| Факторы роста/падения | Блок "факторы" | Базовые аргументы за/против |
| Фундаментальные метрики | Правая колонка | P/E, P/B, капитализация |
| Активность | Частота постов | Высокая активность = интерес |

**Шаг 1.3:** Оценить тональность

```
Анализ 20-30 последних постов:
- Сколько позитивных vs негативных?
- Какие аргументы повторяются?
- Есть ли страх/эйфория?
```

### Шаг 2: MFD — форум трейдеров

**Главная страница форума:**

```
https://mfd.ru/forum/
```

**Важно:** MFD блокирует запросы без User-Agent. Использовать curl:
```bash
curl -s -L -A "Mozilla/5.0 (X11; Linux x86_64) Chrome/120.0.0.0" "https://mfd.ru/forum/"
```

Популярные темы (отображаются на главной):
- Газпром – акции (GAZP)
- Сбербанк – акции (SBER)
- Особенности национальной торговли
- Россети / ФСК (FEES)

### Шаг 3: Banki.ru — финансовый форум

**Раздел рынка ценных бумаг:**

```
https://www.banki.ru/forum/?PAGE_NAME=list&FID=5
```

Разделы форума:
- Рынок ценных бумаг (котировки, новости, обсуждение)
- Ценные бумаги и коллективные инвестиции
- Брокеры

### Шаг 4: Banki.ru — отзывы о банках

**Шаг 4.1:** Определить slug банка

| Тикер | Slug на Banki.ru |
|-------|------------------|
| SBER | `sberbank` |
| TCSG | `tinkoff-bank` |
| VTBR | `vtb` |
| MOEX | `moskovskaya-birzha` |

**Шаг 4.2:** Открыть отзывы

```
https://www.banki.ru/services/responses/bank/{slug}/
```

Примеры:
```
https://www.banki.ru/services/responses/bank/sberbank/
https://www.banki.ru/services/responses/bank/tinkoff-bank/
```

**Шаг 4.3:** Извлечь метрики

| Метрика | Где искать | Что означает |
|---------|------------|--------------|
| Средняя оценка | Рейтинг в шапке | 1-5, общая удовлетворённость |
| Народный рейтинг | Рейтинг + место | Позиция среди банков |
| % решённых проблем | В шапке | Качество сервиса |
| Тональность отзывов | Тексты | Жалобы vs благодарности |

### Шаг 5: Синтез результатов

**Оценка настроения:**

| Оценка | Признаки |
|--------|----------|
| **Очень бычье** | Эйфория, "кGrade в лонг", игнор рисков |
| **Бычье** | Умеренный оптимизм, аргументированные покупки |
| **Нейтральное** | Баланс плюсов/минусов, ожидание |
| **Медвежье** | Опасения, продажа, негативные сценарии |
| **Очень медвежье** | Паника, "всё пропало", пессимизм |

**Контрарный сигнал:**

| Настроение | Контрарное действие |
|------------|---------------------|
| Очень бычье | Рассмотреть продажу (толпа закупилась) |
| Очень медвежье | Рассмотреть покупку (толпа в панике) |
| Нейтральное | Следовать fundamentals |

## Типовые сценарии

### Анализ настроений по акции

```
1. SmartLab: forum/SBER → извлечь 30 постов
2. Оценить: бычье/медвежье
3. Вывод: контрарный сигнал или подтверждение
```

### Репутация банка

```
1. Banki.ru: responses/bank/sberbank
2. Извлечь: средняя оценка, % решённых проблем
3. Прочитать: последние 10 негативных отзывов
4. Вывод: системные проблемы или единичные случаи
```

### Поиск слухов и инсайдов

```
1. SmartLab: форум по тикеру
2. Искать: "слышал", "инсайд", "ожидается"
3. Фильтровать: факты vs мнения
```

## Результат

Структурированный отчёт:

```
## [TICKER] — Social Sentiment

### Источники
- SmartLab: X постов за неделю
- Banki.ru: оценка Z, N отзывов (если банк)

### Тональность
- Бычье: 60%
- Нейтральное: 25%
- Медвежье: 15%

### Ключевые темы
- Тема 1: аргументы за
- Тема 2: аргументы против
- Тема 3: слухи/ожидания

### Контрарный сигнал
[Вывод: следовать или идти против толпы]

### Вывод
[Сводка по настроениям]
```

## Ограничения

- SmartLab: нет пагинации в выдаче (только первая страница)
- MFD: блокирует ботов — нужен curl с User-Agent
- Banki.ru: только для банков/фин. организаций
- Настроения могут быть манипулированы
- Толпа часто ошибается на разворотах
