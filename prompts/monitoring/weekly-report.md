# Weekly Portfolio Report

Выполни еженедельный анализ портфеля.

## Шаги

1. Получить отчёт: `skill portfolio:show`
2. Для позиций с изменением > 5%:
   - Причина: `news news:fetch --ticker $TICKER --days=7`
   - Техника: проанализировать тренд
3. Проверить активные рецепты:
   - Статус каждого
   - Нужно обновление?
4. Рекомендации на неделю:
   - Что держать
   - Что рассмотреть для продажи
   - Что рассмотреть для покупки

## Формат

1. Сводка недели (доходность портфеля)
2. Топ-движения (+/-)
3. Ключевые события
4. Активные рецепты
5. Идеи на неделю

---

Не является инвестиционной рекомендацией.
