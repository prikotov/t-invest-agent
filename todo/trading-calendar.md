# Торговый календарь

## Задача

Реализовать получение торгового календаря через API.

## Зачем

- Определять торговые дни и выходные
- Знать часы торгов для разных инструментов (акции, фьючерсы, облигации)
- Учитывать праздники
- Корректно определять T+ режим расчётов

## Источники

### T-Invest API (приоритет)

Метод `TradingSchedules` в сервисе Instruments:
- https://tinkoff.github.io/investAPI/instruments/#tradingschedules
- Возвращает: `is_trading_day`, `start_time`, `end_time`, аукционы открытия/закрытия, вечерняя сессия, клиринг
- Фильтрация по бирже (`exchange`) и периоду (`from`, `to`)

### MOEX ISS API (альтернатива)

- `/engines/stock/trading.json` — расписание торгов
- Нужен если работаем без токена T-Invest

## Формат

CLI команда в t-invest-core:
```bash
./bin/skill schedule [--exchange MOEX] [--date YYYY-MM-DD]
```

Вывод:
- Торговый день или выходной
- Часы торгов (основная сессия, вечерняя, аукционы)
- Время клиринга

## Приоритет

Medium — полезно для агента (проверка "сегодня торги?")
